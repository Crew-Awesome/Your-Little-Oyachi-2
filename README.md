# Your Little Oyachi 2

This project is a sequel to the original Your Little Oyachi game.

It is a cozy in-browser Three.js pet experience where you can interact with Oyachi, move between rooms, and keep evolving the world with new features and visuals.

Original game:
- https://github.com/Crew-Awesome/Your-Little-Oyachi-1
